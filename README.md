# منصة توثيق مسؤوليات خدمات الموارد البشرية

نموذج تشغيلي احترافي باللغة العربية لإدارة مسؤوليات خدمات الموارد البشرية وبناء ملفات إجراءات Departmental Policy & Procedures لكل مسؤولية.

## التشغيل

```powershell
node server.mjs
```

ثم افتح:

```text
http://localhost:4173
```

لا يحتاج النموذج إلى تثبيت حزم. البيانات محفوظة في `localStorage` لأغراض التجربة، ومخطط قاعدة البيانات المقترح موجود في `prisma/schema.prisma`.

## ما تم تضمينه

- صفحة مسؤوليات كاملة مع فلاتر متقدمة.
- صفحة إدخال بنظام خطوات Wizard.
- حساب تلقائي لنسبة الاكتمال والوزن والعبء.
- داشبورد تنفيذي مع بطاقات ورسوم SVG.
- رفع Excel/CSV مع معاينة وأخطاء قبل الحفظ.
- صلاحيات تجريبية Admin و HR Specialist.
- Audit Log مبسط.
- بيانات Demo جاهزة.
